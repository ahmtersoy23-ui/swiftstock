OMS Entegrasyon

OMS ↔ WMS Entegrasyon API

OMS karar verir, WMS yurutur. Iki bagimiz katman, standart API ile konusur.

OMS → WMS (Talimat)

GET /oms/stock

IWASKU + depo bazinda kullanilabilir stok sorgulama. Fiziksel - rezerve = net.

POST /oms/reserve

Siparis icin stok rezervasyonu. Otomatik expiry destegi.

POST /oms/pick-request

Siparis bazli picking talimati. Oncelik + lokasyon siralamali pick listesi.

WMS → OMS (Bildirim)

Webhook: STOCK_CHANGE

Her IN/OUT/TRANSFER/COUNT sonrasi guncel stok bildirimi.

Webhook: SHIPMENT_STATUS

Sevkiyat durumu degisikliginde (CLOSED, SHIPPED) bildirim.

Webhook: ORDER_STATUS

Picking tamamlandi, siparis durumu degisti bildirimi.

Webhook retry: 3 deneme, exponential backoff | wms_webhook_logs ile audit trail
