OMS Entegrasyon

OMS ↔ WMS Entegrasyon API

OMS karar verir, WMS yürütür. İki bağımsız katman, standart API ile konuşur.

OMS → WMS (Talimat)

GET /oms/stock

IWASKU + depo bazında kullanılabilir stok sorgulama. Fiziksel − rezerve = net.

POST /oms/reserve

Sipariş için stok rezervasyonu. Otomatik expiry desteği.

POST /oms/pick-request

Sipariş bazlı picking talimatı. Öncelik + lokasyon sıralamalı pick listesi.

WMS → OMS (Bildirim)

Webhook: STOCK_CHANGE

Her IN/OUT/TRANSFER/COUNT sonrası güncel stok bildirimi.

Webhook: SHIPMENT_STATUS

Sevkiyat durumu değişikliğinde (CLOSED, SHIPPED) bildirim.

Webhook: ORDER_STATUS

Picking tamamlandı, sipariş durumu değişti bildirimi.

Webhook retry: 3 deneme, exponential backoff | wms_webhook_logs ile audit trail
