Entegrasyon API

OMS ↔ WMS API Hazır ✅

OMS → WMS (Talimat)

GET /oms/stock

IWASKU + depo bazında kullanılabilir stok. Fiziksel − rezerve = net.

POST /oms/reserve

Sipariş için stok rezervasyonu. Otomatik expiry desteği.

DELETE /oms/reserve/:id

Rezervasyon iptal (sipariş iptali).

POST /oms/pick-request

Sipariş bazlı picking talimatı. Lokasyon sıralamalı pick listesi.

WMS → OMS (Webhook)

STOCK_CHANGE

Her IN/OUT/TRANSFER/COUNT sonrası güncel stok bildirimi.

SHIPMENT_STATUS

Sevkiyat durumu: CLOSED / SHIPPED.

ORDER_STATUS

Picking tamamlandı, sipariş durumu değişti.

Altyapı

Webhook kayıt/silme/test | 3 deneme exponential backoff | Audit log

Wisersell OMS

Wisersell — OMS Katmanı

Wisersell sipariş yönetimini üstlenir. SwiftStock WMS ile API üzerinden entegre çalışır.

Rol Dağılımı

Wisersell (OMS)Sipariş alma, allocation, kanal stok push, kargo, takip no
SwiftStock (WMS)Fiziksel stok, depo operasyonları, picking, sevkiyat

Entegrasyon

  • → WMS Stok kontrol, rezervasyon, picking talimatı
  • ← WMS Stok değişikliği, sevkiyat durumu webhook
  • Sync WMS tek doğru kaynak (single source of truth)
